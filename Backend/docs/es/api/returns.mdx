description: "Consulta los registros de devoluciones de SmartPyme filtrados por rango de fechas. Cada registro incluye campos de cabecera, ID de la venta original, líneas de detalle y resúmenes agregados."
---

Los endpoints de Devoluciones exponen todas las transacciones de devolución de ventas registradas en SmartPyme, permitiéndote conciliar ajustes de inventario, auditar la actividad de reembolsos a clientes y rastrear la mercadería devuelta a nivel de línea de detalle. Cada registro de devolución se vincula con su venta original mediante `id_venta` e incluye un desglose completo en `detalles` de los productos devueltos. Todas las solicitudes deben incluir tu API key en el encabezado `Authorization`.

---

## GET /returns

Devuelve una lista paginada de registros de devoluciones de ventas. Filtra por rango de fechas para acotar los resultados a un período específico. Cada registro incluye la información de cabecera de la devolución y un arreglo `detalles` con los productos individuales que fueron devueltos.

<CodeGroup>

```bash cURL
curl --request GET \
  --url "https://api.smartpyme.site/api/external/v1/returns?fecha_inicio=2025-01-01&fecha_fin=2025-01-31&per_page=50" \
  --header "Authorization: Bearer {api_key}"
```

```python Python
import requests

url = "https://api.smartpyme.site/api/external/v1/returns"
headers = {"Authorization": "Bearer {api_key}"}
params = {
    "fecha_inicio": "2025-01-01",
    "fecha_fin": "2025-01-31",
    "per_page": 50,
}

response = requests.get(url, headers=headers, params=params)
print(response.json())
```

```javascript JavaScript
const response = await fetch(
  "https://api.smartpyme.site/api/external/v1/returns" +
    "?fecha_inicio=2025-01-01&fecha_fin=2025-01-31&per_page=50",
  {
    method: "GET",
    headers: { Authorization: "Bearer {api_key}" },
  }
);
const data = await response.json();
console.log(data);
```

</CodeGroup>

### Parámetros de consulta

<ParamField query="fecha_inicio" type="string">
  Fecha de inicio para filtrar devoluciones, en formato `Y-m-d` (por ejemplo, `2025-01-01`). Úsala junto con `fecha_fin` para definir un rango de fechas.
</ParamField>

<ParamField query="fecha_fin" type="string">
  Fecha de fin para filtrar devoluciones, en formato `Y-m-d` (por ejemplo, `2025-01-31`). Úsala junto con `fecha_inicio` para definir un rango de fechas.
</ParamField>

<ParamField query="page" type="integer">
  Número de página a obtener. Su valor por defecto es `1`.
</ParamField>

<ParamField query="per_page" type="integer">
  Cantidad de registros por página. Acepta valores entre `1` y `200`. Su valor por defecto es `100`.
</ParamField>

## GET /returns/\{id\}

Recupera un único registro de devolución por su ID numérico. La respuesta devuelve el mismo objeto Devolución completo descrito anteriormente, incluido el arreglo `detalles` con las líneas devueltas.

### Parámetro de ruta

<ParamField path="id" type="integer" required>
  El identificador numérico único del registro de devolución que deseas obtener.
</ParamField>

<CodeGroup>

```bash cURL
curl --request GET \
  --url "https://api.smartpyme.site/api/external/v1/returns/789" \
  --header "Authorization: Bearer {api_key}"
```

```python Python
import requests

return_id = 789
url = f"https://api.smartpyme.site/api/external/v1/returns/{return_id}"
headers = {"Authorization": "Bearer {api_key}"}

response = requests.get(url, headers=headers)
print(response.json())
```

```javascript JavaScript
const returnId = 789;
const response = await fetch(
  `https://api.smartpyme.site/api/external/v1/returns/${returnId}`,
  {
    method: "GET",
    headers: { Authorization: "Bearer {api_key}" },
  }
);
const data = await response.json();
console.log(data);
```

</CodeGroup>

La respuesta envuelve el objeto Devolución bajo una clave `data` con `"success": true`. Todos los campos, incluido `detalles`, son idénticos a la estructura documentada en la respuesta de [GET /returns](#get-returns) anterior.

## GET /returns/summary

Devuelve estadísticas agregadas para tu actividad de devoluciones sobre un rango de fechas opcional. Usa este endpoint para medir el volumen y el valor de la mercadería que se está devolviendo, identificar períodos con altas devoluciones y cuantificar el impacto financiero de las devoluciones sobre tus ingresos.

<CodeGroup>

```bash cURL
curl --request GET \
  --url "https://api.smartpyme.site/api/external/v1/returns/summary?fecha_inicio=2025-01-01&fecha_fin=2025-01-31" \
  --header "Authorization: Bearer {api_key}"
```

```python Python
import requests

url = "https://api.smartpyme.site/api/external/v1/returns/summary"
headers = {"Authorization": "Bearer {api_key}"}
params = {
    "fecha_inicio": "2025-01-01",
    "fecha_fin": "2025-01-31",
}

response = requests.get(url, headers=headers, params=params)
print(response.json())
```

```javascript JavaScript
const response = await fetch(
  "https://api.smartpyme.site/api/external/v1/returns/summary" +
    "?fecha_inicio=2025-01-01&fecha_fin=2025-01-31",
  {
    method: "GET",
    headers: { Authorization: "Bearer {api_key}" },
  }
);
const data = await response.json();
console.log(data);
```

</CodeGroup>

### Parámetros de consulta

<ParamField query="fecha_inicio" type="string">
  Fecha de inicio del período de resumen, en formato `Y-m-d`. Omítela para incluir todos los registros de devoluciones desde el inicio.
</ParamField>

<ParamField query="fecha_fin" type="string">
  Fecha de fin del período de resumen, en formato `Y-m-d`. Omítela para incluir todos los registros hasta la fecha actual.
</ParamField>

### Ejemplo de respuesta

```json
{
  "success": true,
  "data": {
    "cantidad_devoluciones": 45,
    "total_devuelto": 6750.00,
    "total_iva": 607.50,

    "promedio_devolucion": 150.00,

      { "tipo": "Devolucion", "cantidad": 45, "total": 6750.00 }
    ]

  "meta": {
    "empresa": "Mi Empresa S.A.",
    "timestamp": "2025-01-31T23:59:00Z",
    "filters_applied": {
      "fecha_inicio": "2025-01-01",
      "fecha_fin": "2025-01-31"
    }
  }
}
```

### Campos de la respuesta

<ResponseField name="data.cantidad_devoluciones" type="integer">
  Número total de transacciones de devolución procesadas dentro del período solicitado.
</ResponseField>

<ResponseField name="data.total_devuelto" type="number">
  Valor monetario bruto de todas las devoluciones en el período, incluyendo impuestos.
</ResponseField>

<ResponseField name="data.total_iva" type="number">
  Monto total de impuesto incluido en todos los registros de devolución del período.
</ResponseField>

<ResponseField name="data.total_descuentos" type="number">
  Suma de todos los descuentos aplicados a las líneas devueltas en el período.
</ResponseField>

<ResponseField name="data.promedio_devolucion" type="number">
  Valor promedio por transacción de devolución, calculado como `total_devuelto / cantidad_devoluciones`.
</ResponseField>

<ResponseField name="data.devoluciones_por_tipo" type="array">
  Desglose del conteo y valor total de devoluciones agrupado por tipo de documento.

  <Expandable title="Campos de devoluciones_por_tipo">
    <ResponseField name="tipo" type="string">
      El tipo de documento de devolución (por ejemplo, `Devolucion`).
    </ResponseField>
    <ResponseField name="cantidad" type="integer">
      Número de registros de devolución con este tipo de documento.
    </ResponseField>
    <ResponseField name="total" type="number">
      Valor total combinado de las devoluciones con este tipo de documento.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="meta.empresa" type="string">
  Nombre visible de la empresa asociada a tu API key.
</ResponseField>

<ResponseField name="meta.timestamp" type="string">
  Marca de tiempo en formato ISO 8601 que indica cuándo se generó el resumen.
</ResponseField>

<ResponseField name="meta.filters_applied" type="object">
  Eco de los filtros de fecha utilizados para calcular el resumen, útil para confirmar los parámetros de la solicitud.
</ResponseField>
