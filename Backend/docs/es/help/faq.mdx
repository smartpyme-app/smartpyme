sidebarTitle: "Preguntas frecuentes"
description: "Respuestas a las preguntas más comunes sobre SmartPyme acerca de cuentas, facturación, facturación electrónica, integraciones de API y gestión de datos."
---

Ya sea que estés comenzando o gestionando una operación consolidada, a continuación encontrarás las respuestas a las preguntas más comunes sobre SmartPyme. Si no ves lo que buscas, escríbele a nuestro equipo de soporte a soporte@smartpyme.com.

<AccordionGroup>
  <Accordion title="¿Cómo restablezco mi contraseña?">
    Ve a la página de inicio de sesión en [app.smartpyme.site](https://app.smartpyme.site) y haz clic en **Olvidé mi contraseña**. Ingresa el correo asociado a tu cuenta y sigue el enlace de restablecimiento enviado a tu bandeja de entrada.
  </Accordion>

  <Accordion title="¿Puedo tener varias sucursales?">
    Sí. SmartPyme soporta varias sucursales. Agrégalas en **Configuración → Sucursales**. Cada sucursal opera de forma independiente con sus propios usuarios, cajas y bodegas.
  </Accordion>

  <Accordion title="¿Cómo agrego más usuarios?">
    Ve a **Configuración → Usuarios** y haz clic en **Nuevo usuario**. Ingresa los datos del usuario, asígnale un rol —**admin**, **cajero**, **vendedor** o **supervisor**— y selecciona la sucursal a la que pertenece.
  </Accordion>

  <Accordion title="¿Se soporta la facturación electrónica (DTE)?">
    Sí, SmartPyme soporta la facturación electrónica (DTE) para El Salvador. Necesitas credenciales válidas del Ministerio de Hacienda para activar esta función. Consulta la [guía de facturación electrónica](/features/electronic-invoicing) para todos los pasos de configuración.
  </Accordion>

  <Accordion title="¿Puedo importar mis productos y clientes existentes?">
    Sí. Para importar productos, ve a **Inventario → Productos → Importar** y carga un archivo Excel usando la plantilla proporcionada. Para importar clientes, ve a **Ventas → Clientes → Importar** y sigue el mismo proceso con la plantilla de clientes.
  </Accordion>

  <Accordion title="¿Qué métodos de pago soporta SmartPyme?">
    SmartPyme incluye los siguientes métodos de pago integrados:

    - **Efectivo**
    - **Tarjeta**
    - **Transferencia** (transferencia bancaria)
    - **Cheque**
    - **Crédito**

    También puedes agregar métodos de pago personalizados en **Configuración → Métodos de pago**.
  </Accordion>

  <Accordion title="¿Cómo obtengo mi API Key para la API externa?">
    Ve a **Configuración → Mi cuenta** y abre la pestaña **Integraciones**. Tu API Key se muestra allí. Mantenla confidencial — trátala como una contraseña.
  </Accordion>

  <Accordion title="¿Qué pasa si excedo el límite de uso de la API?">
    Tu solicitud devuelve una respuesta **HTTP 429 Too Many Requests**. El límite se restablece al inicio de la siguiente hora. Mientras esperas, evita reintentar la misma solicitud en bucle. Consulta la [documentación de límites de uso](/api/rate-limits) para conocer los detalles sobre los límites y las estrategias de optimización.
  </Accordion>

  <Accordion title="¿Puedo conectar mi tienda de WooCommerce?">
    Sí. Configura el webhook en **Configuración → Mi cuenta → Integraciones** usando tu token de SmartPyme. Consulta la [guía de integración con WooCommerce y Shopify](/integrations/woocommerce-shopify) para las instrucciones paso a paso.
  </Accordion>

  <Accordion title="¿Cómo cancelo mi suscripción?">
    Contacta al soporte de SmartPyme en [soporte@smartpyme.com](mailto:soporte@smartpyme.com) para solicitar la cancelación. Tus datos permanecen totalmente accesibles hasta el final de tu período de facturación actual.
  </Accordion>
</AccordionGroup>
