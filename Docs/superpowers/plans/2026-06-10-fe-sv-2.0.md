# FE El Salvador 2.0 — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Cumplir Normativa DTE v2.0 del MH con los **11 tipos DTE** operativos end-to-end en SmartPyme antes del 1-dic-2026.

**Architecture:** Mantener el flujo actual (builder `MH*` → firma externa → `MhDteProxyController` → MH). Introducir `DteSchemaVersionResolver` con feature flag por empresa, tabla `dte_eventos` para Retorno/Operaciones Especiales, y servicios de validación de catálogos. Deprecar generación vía `MH.php` legacy; un solo path en `ElSalvadorDteService`.

**Tech Stack:** Laravel 8+/PHP 8.3, Angular 15, MySQL, firmador EC2, API MH (`apitest.dtes.mh.gob.sv` / prod).

**Spec de referencia:** [`Docs/PLAN-FE-SV-2.0.md`](../../Docs/PLAN-FE-SV-2.0.md)

### Task 1: Resolver + migración empresa

**Files:**

- [ ] **Step 2: Resolver**

```php
<?php
namespace App\Support\FacturacionElectronica\ElSalvador;

class DteSchemaVersionResolver
{
    /** Versiones estructurales MH por tipo cuando empresa está en v2 */
    private const V2_VERSIONS = [
        '01' => 2,
        '03' => 4, // confirmar contra schema MH v2.0 publicado
        '04' => 2,
        '05' => 4,
        '06' => 4,
        '07' => 2,
        '08' => 2,
        '09' => 2,
        '11' => 2,
        '14' => 2,
        '15' => 2,
    ];

    public function resolve(string $tipoDte, ?string $empresaSchema = null): int
    {
        $mode = $empresaSchema ?? config('fe_sv.schema_default', 'legacy');
        if ($mode !== 'v2') {
            return $this->legacyVersion($tipoDte);
        }
        return self::V2_VERSIONS[$tipoDte] ?? 1;
    }

    private function legacyVersion(string $tipoDte): int
    {
        return match ($tipoDte) {
            '03', '05', '06' => 3,
            default => 1,
        };
    }
}
```

> **Nota:** Los enteros de `V2_VERSIONS` deben alinearse con los schemas descargados del MH en PR-01; ajustar tras diff real.
